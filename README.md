## rock-paper-scissors

## Description

Acceptance Criteria:

- As a user, I want to play Rock, Paper, Scissors against an automated opponent.

- As a user, I can enter R, P, or S to signify my choice of rock, paper, or scissors.

- As a user, I expect the computer to choose R, P, or S in return.

- As a user, I want the option to play again whether I win or lose.

- As a user, I want to see my total wins, ties, and losses after each round.

## Installation

Download a source-code editor, such as VS Code and play index.html on default browser.

## Usage

Open Index.html file on default browser. Answer prompt questions. Prompt will tell you who won and keep track of the score. Check console to confirm.
